# Online-Examination-System
Online Examination System using PHP &amp; MySQL (Admin &amp; Student Panel)


🎓 Diploma Final Year Project  
💻 First Academic Web Development Project

---

## Project Description

The Online Examination System is a web-based application developed as part of my Diploma Final Year Project.  
This is my first complete academic project, where I designed and implemented both the frontend and backend using PHP and MySQL.

The system allows educational institutions to conduct examinations online, manage students, evaluate results, and reduce manual work.

---

## Technologies Used

- Frontend: HTML, CSS, Bootstrap, JavaScript  
- Backend: PHP  
- Database: MySQL  
- Server: Apache (XAMPP)

---

## User Roles

### Admin Panel
- Secure admin login
- Manage students, departments, categories, and subjects
- Create and manage examinations
- Add questions (MCQs and fill in the blanks)
- Set exam duration, expiry date, and re-exam timing
- View exam results and performance statistics
- Post notices and announcements
- Manage admin profile

### Student Panel
- Secure student login
- View available examinations
- Attend exams online
- View exam results
- View students in the same class/category
- Update profile details

---

## Database Overview

Main tables used:
- tbl_users
- tbl_exams
- tbl_questions
- tbl_results
- tbl_departments
- tbl_categories
- tbl_subjects

---

## How to Run the Project

1. Install XAMPP  
2. Copy the project folder to `C:\xampp\htdocs\`  
3. Start Apache and MySQL  
4. Create a database named `oes_db`  
5. Import the SQL file using phpMyAdmin  
6. Configure database in `system/database/config.php`  
7. Open in browser: `http://localhost/online_exam`
